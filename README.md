# logolotse-skills

Operative Paperclip-Skills für das LogoLotse Agent-Team.

## Was sind Skills?

Skills definieren das operative Verhalten der LogoLotse-Agenten — wie sie eskalieren, formulieren, entscheiden und handeln. Jeder Skill ist ein Ordner mit einer `SKILL.md` und optionalen Begleitdateien.

## Skills

| Skill | Beschreibung |
|---|---|
| `logolotse-cd` | Corporate Design — Marken-, UI- und Microcopy-Regeln |
| `logolotse-verfassung` | Unternehmens-Verfassung v1.3 — Prinzipien, Eskalation, Governance |
| `logolotse-mission` | Firmenbeschreibung, Mission und Vision |
| `logolotse-logo` | Logo-Verwendung und Varianten |
| `logolotse-webserver-deployment` | VPS-Deployment-Prozess |
| `logolotse-logopaedie-artefakte` | Domain-Wissen Schullogopädie |
| `logolotse-agent-optimizer` | Agent-Optimierungs-Prozesse |

## Single Source of Truth

Dieses Repo ist die einzige Quelle der Wahrheit für alle Skills.
Das Paperclip-Lokal-Verzeichnis ist eine Working Copy — niemals direkt editieren.

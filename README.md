# logolotse-skills

Oeffentliche Paperclip-Skills fuer das LogoLotse Agent-Team.

## Was sind Skills?

Skills definieren das operative Verhalten der LogoLotse-Agenten — wie sie
eskalieren, formulieren, entscheiden und handeln. Jeder Skill ist ein Ordner
mit einer SKILL.md und optionalen Begleitdateien.

## Skills in diesem Repo

| Skill | Beschreibung |
|---|---|
| logolotse-cd | Corporate Design — Marken-, UI- und Microcopy-Regeln |
| logolotse-verfassung | Unternehmens-Verfassung v1.3 — Prinzipien, Eskalation, Governance |
| logolotse-mission | Firmenbeschreibung, Mission und Vision |
| logolotse-logopaedie-research | DLV-Research, Produktableitung Schullogopaedie |
| logolotse-agent-optimizer | Agent-Optimierungs-Prozesse |

## Nicht-oeffentliche Skills

Skills mit Infrastruktur-Details, Credentials-Referenzen oder operativ
sensiblen Inhalten liegen im privaten logolotse-knowledge Repository
(skills/) und werden via lokalem Pfad in Paperclip eingebunden.

## Single Source of Truth

Dieses Repo ist die einzige Quelle der Wahrheit fuer die hier abgelegten Skills.
Das Paperclip-Lokal-Verzeichnis ist eine Working Copy — niemals direkt editieren.

## Verwendung in Paperclip

Skills werden via GitHub-URL importiert:

  https://github.com/LogoLotse/logolotse-skills

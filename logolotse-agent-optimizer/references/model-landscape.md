# Model Landscape Rechercheleitfaden

## Ziel

Eine vergleichbare, aktuelle Sicht auf führende Modelle pro Anbieter für Team-Entscheidungen.

## Anbieter

- Anthropic
- OpenAI
- Google
- Mistral
- xAI
- Open-Weights (z. B. Llama, Qwen, DeepSeek)

## Kategorien

- Frontier Reasoning
- Mid-Tier General Purpose
- Lightweight/Cost-Optimized
- Code-Spezialist

## Pflichtquellen

- Offizielle Modell-Dokumentation/Release-Notes je Anbieter
- Offizielle Pricing-Seiten je Anbieter
- Mindestens ein neutraler Benchmark-Vergleich pro Kategorie

## Datenpunkte je Modell

- Modellname und Release-/Update-Datum
- Stärken (Reasoning, Coding, Tool-Use, Context)
- Schwächen / bekannte Einschränkungen
- Preisinput und Preisoutput
- Latenz-/Throughput-Hinweise (falls verfügbar)
- Verfügbarkeit im aktuellen Agent-Setup

## Qualitätskriterien

- Nur Quellen mit Datum der letzten Aktualisierung verwenden
- Veraltete Modellnamen explizit ausschliessen
- Aussagen immer mit URL und Datum belegen

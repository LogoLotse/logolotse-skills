# Evaluation Matrix je Agent

## Scoring (0-5 je Dimension)

- Modell-Fit: 35%
- Aufgaben-Fit (Capabilities): 25%
- Operative Taktung (Heartbeat): 20%
- Skill-Abdeckung: 20%

Gesamtscore = gewichteter Mittelwert.

## Dimensionen

### 1) Modell-Fit (35%)

- 5: Exzellenter Fit, keine bessere Alternative mit relevantem Vorteil
- 3: Solider Fit, kleinere Verbesserungsmöglichkeiten
- 1: Klarer Mismatch oder deutlich ineffizient

### 2) Aufgaben-Fit / Capabilities (25%)

- 5: Vollständig, aktuell, präzise
- 3: Teilweise unpräzise oder leicht veraltet
- 1: Klar veraltet oder falscher Scope

### 3) Heartbeat-Taktung (20%)

- 5: 40-70% produktive Runs
- 3: 25-39% oder 71-80% produktive Runs
- 1: <25% oder >80% mit klaren Warte-/Stau-Signalen

### 4) Skill-Abdeckung (20%)

- 5: Alle nötigen Skills vorhanden, keine Altlasten
- 3: Kleine Lücken oder ungenutzte Skills
- 1: Kritische Lücken oder deutlich veraltete Skill-Liste

## Entscheidungsregeln

- Score >= 4.2: Keine Änderung empfohlen
- Score 3.3-4.1: Gezielte kleine Anpassung möglich
- Score < 3.3: Änderungsvorschlag vorbereiten (mit Quellen)

## Pflichtbelege bei Änderung

- Benchmark- oder Qualitätsbeleg (URL + Datum)
- Pricing-Beleg (URL + Datum)
- Erwarteter Impact (Qualität, Kosten, Durchlaufzeit)

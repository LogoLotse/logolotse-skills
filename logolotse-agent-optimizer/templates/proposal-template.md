# Team-Optimierungsvorschlag (LogoLotse)

## Metadaten

- Datum: YYYY-MM-DD
- Autor-Agent: <agent-id>
- Review-Typ: Monatlich | Quartal
- Vergleichszeitraum: <Monat/Quartal>

## 1. Management Summary

- Empfohlene Änderungen (kurz)
- Nicht-Änderungen (kurz)
- Hauptrisiken / Annahmen

## 2. Modell-Landschaft seit letztem Review

- Wichtige Veränderungen je Anbieter
- Relevanz für LogoLotse-Agenten
- Quellen (URL + Datum)

## 3. Agenten-Bewertung und Vorschläge

### Agent: <name / id>

- Aktueller Zustand
- Score (gesamt + 4 Dimensionen)
- Vorschlag (bleibt | ändern)
- Begründung mit Quellen
- Kosteneffekt (USD/Monat)
- Risiko / Rollback-Idee

(Abschnitt pro betroffenem Agent wiederholen)

## 4. Unveränderte Agenten

- <Agent> bleibt, weil ...

## 5. Organisations-Analyse (nur Quartal)

- Signal-Befunde (Split/Merge/Hire/Terminate)
- Empfehlung (eine oder keine)
- Offene Board-Entscheidung

## 6. Approval-Plan

- Geplante Approval-Requests (ein Request pro Änderung)
- Reihenfolge
- Abhängigkeiten

## 7. Abschlussdaten

- Erwarteter Netto-Kosteneffekt gesamt
- Datum nächster Review
- Offene Fragen an den Board
